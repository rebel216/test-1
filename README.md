# Test-jest

> An exercise to test different functions using jest.

## Built With

- JavaScript
- JEST

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- Any code editor
- Terminal

### Install

- npm install --save-dev jest
- jest --init

### Run tests

- npm test
